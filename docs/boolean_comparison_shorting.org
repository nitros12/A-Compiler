* Boolean Comparisons
  Short circuiting, in the expression ~(a && b)~, ~b~ will only be evalated if ~a~ was truthy.

** Implementation ~(a || b)~
#+BEGIN_SRC asm
  r1 <- a
  cmp r1 0
  jne end
  r2 <- b
  r1 <- r2
  end:
#+END_SRC

** Implementation ~(a && b)~
#+BEGIN_SRC asm
  r1 <- a
  cmp r1 0
  jeq end
  r2 <- b
  r1 <- r2
  end:
#+END_SRC
